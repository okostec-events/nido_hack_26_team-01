## Project Pitch (2 sentences max)

_What does your project do, and why does it matter?_

Impact is a platform that turns real-world human actions into measurable environmental and community impact in real time.
It allows companies and municipalities to track, prove, and communicate their impact with credibility instead of relying on delayed or abstract reporting.

Not deployed (local demo used for presentation)

## How to run the project

Clone the repository
Run npm install
Run npm run dev and open http://localhost:3000

## What problem does it solve?

Companies and municipalities are under pressure to demonstrate real environmental and social impact, but their current data is often delayed, hard to verify, and disconnected from actual human actions.
This creates a gap in trust and forces organizations to rely on last-minute, costly efforts to meet compliance or improve public perception.
Impact solves this by connecting individual actions directly to measurable, real-time metrics, making impact visible, credible, and continuous.

## Anything the judges should know?

This project is designed as a mobile-first MVP focused on demonstrating a clear, reliable demo flow using mock data.
The core value lies in the system: transforming human actions into measurable impact that organizations can track, scale, and use to build trust while reducing reactive compliance costs.
